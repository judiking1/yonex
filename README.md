# yonex

<h1>
  Yonex 이벤트 페이지
  <br>
  (23.03.24 ~ 23.04.12)
</h1>

<p>
  CSS + jQuery로 만들었습니다.
  <br>
  메뉴 호버시 서브 메뉴 탭과 화면 우측 하단에는 추가 탭이 나옵니다.<br>
  전체 영상은 컨트롤러를 통해 제어가 가능합니다.<br>
  미디어쿼리를 활용해 태블릿 버전을 추가했습니다.<br>
  해당 페이지는 태블릿 아이콘을 클릭하여 확인 가능합니다. <br>
  태블릿 화면에서 다시 pc아이콘을 클릭하면 PC버전으로 전환됩니다.
</p>
